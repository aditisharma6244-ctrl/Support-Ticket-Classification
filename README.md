#  Support Ticket Classification using NLP

##  Project Overview

This project focuses on classifying customer support tickets into different categories such as Billing Inquiry, Technical Issue, Refund Request, and Cancellation Request using Natural Language Processing (NLP) and Machine Learning techniques.

##  Objective

To build a machine learning model that can automatically classify customer support tickets based on their text content.

##  Technologies Used

* Python
* Pandas
* Scikit-learn
* Natural Language Processing (NLP)
* TF-IDF Vectorization

##  Steps Performed

1. Data Loading and Exploration
2. Data Cleaning and Preprocessing
3. Text Processing using NLP
4. Feature Extraction using TF-IDF
5. Model Training using Naive Bayes
6. Model Evaluation using Accuracy, Precision, Recall, and F1-score

##  Model Performance

The model was evaluated using standard metrics like accuracy, precision, recall, and F1-score.
Due to repetitive and similar text in the dataset, the accuracy is moderate.
